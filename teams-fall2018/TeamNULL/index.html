IDEA

The definitive hub of information dedicated to expediting the game buying process.

GAMEFAX is designed to be a tool to help gamers spend less time deciding on what they want to play, and spend more time playing. There are so many games out on the market that it can be hard to find the next best experience to purchase. Money is important to everyone and we always want to make sure that we are making the right decision.

That's where GAMEFAX comes in. We have designed a database system to collect all the information about any given title for you, so that you don't have to. Instead of spending countless time scouring the web for information about a game, let GAMEFAX do all the tedious work for you.
